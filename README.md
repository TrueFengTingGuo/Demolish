# Demolish
2d Platform game

# Features
In game tasks:
- Pushing cargo.
-	Kill all enemies within a time limit.
-	Holding Point
Type of Enemy:
- Ranger
- Warrior
- Heavy armor
-	Giant
-	Boss
Achievement:
-	Finishing Task in a time limit
-	Kill enemies while in air.
-	Etc.
Weapon:
-	Arrow
-	Chain
-	Sword
Animation:
-	Extra animations of characters that needs to draw by myself.
Sacore:
-	Score system
-	Timer
UI:
-	Menu
-	In-game UI
-	Level Picking
-	Game Setting(maybe)
